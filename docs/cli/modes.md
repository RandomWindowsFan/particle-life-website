# Modes

[[toc]]

## Color Modes

Defines with what colors particles should be rendered.

| Color Mode |   |
|---|---|
| 0 | black & white |
| 1 (default) | color |

## Position Modes

Defines how the positions of new particles are distributed.

| Position Mode |   |
|---|---|
| 1 | uniform |
| 2 (default) | centered |
| 3 | horizontal |
| 4 | spiral |

## Attraction Modes

Defines how the values of the attraction matrix should be initialized.

| Attraction Mode |   |
|---|---|
| 1 (default) | random |
| 2 | snakes |
